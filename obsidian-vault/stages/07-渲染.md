# 阶段七：渲染（HTML → PNG）

> 将每页的 LayoutSpec 渲染为 HTML，再用 Playwright headless Chromium 截图为 1920×1080 PNG。

## 触发

在 `_compose_render_worker()` 中，编排完成后自动执行：

```python
slides = await compose_all_slides(...)
await render_and_screenshot(project_id, slides, db)
  → render_slide_html(slide, theme, assets)     # for each slide
  → screenshot_slides_batch(html_list)          # Playwright 批量截图
```

或通过 API 单独触发：
```

→ api/routers/render.py
```

## 截图批处理（`render/exporter.py`）

### `screenshot_slides_batch()` 流程

```python
# render/exporter.py line 81
async def screenshot_slides_batch(slides_html: list[str], output_dir: Path):
    async with async_playwright() as pw:
        browser = await pw.chromium.launch(headless=True)
        semaphore = asyncio.Semaphore(4)     # 最多 4 并发 tab
        tasks = [_screenshot_one_page(browser, html, semaphore) for html in slides_html]
        results = await asyncio.gather(*tasks)
```

每页流程：
```
1. 写 HTML 到临时文件
2. Playwright Page.goto(file:// URL, wait_until="networkidle")
3. await page.wait_for_timeout(500)  → 等待字体加载
4. page.screenshot(type="png", clip={0,0,1920,1080})
5. 保存到 tmp/e2e_output/slides/slide_{N:02d}.png
6. os.unlink(temp HTML)
```

### 截图参数

| 参数 | 值 |
|------|-----|
| 宽度 | 1920 px |
| 高度 | 1080 px |
| 并发 tab | 最多 4 |
| 图片格式 | PNG |
| 等待策略 | `networkidle` + 500ms |

## 降级机制
