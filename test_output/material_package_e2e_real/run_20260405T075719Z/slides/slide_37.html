方案二 · 设计注释
  • 工业线条语言：折板屋面以 15°–30° 倾角重复叠合，立面形成强烈水平向肌理，致敬工业厂房美学
  • 天窗采光策略：北向锯齿天窗全天提供稳定漫射光，展厅照度均匀度≥0.7，减少人工照明依赖约40%
  • 材质体系：主体采用耐候钢板（外立面）+ 清水混凝土（基座）+ 穿孔铝板（遮阳格栅），三层材质形成粗—中—细质感递进
  • 街道界面：底层局部架空形成灰空间，退让2.5m形成骑楼式檐廊，激活沿街公共活动
