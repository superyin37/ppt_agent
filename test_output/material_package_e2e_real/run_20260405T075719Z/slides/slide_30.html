方案一 · SCHEME A

太和归真

道法自然，曲线即山脊，屋面即大地，建筑归于天地之间。

形态母题取自武当山太和殿屋脊曲线，以连续起伏的覆土屋面回应山地地貌，建筑消隐于自然之中。被动通风系统沿屋脊走向布置，利用热压差实现自然换气，无需机械驱动。

  • 形态母题：太和殿屋脊曲线，连续弧形屋面与山体轮廓融合
  • 覆土屋面：种植层厚度 600mm，隔热蓄冷，降低建筑能耗约 30%
  • 被动通风：屋脊设导风槽，热压通风换气次数≥6次/小时
  • 空间叙事：入口—庭院—殿堂三段式序列，呼应道教礼仪动线
