小体量·大生态：Trado公厕

MOL Arquitectura · 西班牙 · 2018 · 56㎡

  • 被动通风设计：利用建筑形体开口引导自然穿堂风，全年无需机械通风
  • 在地石材围护：采用当地片岩砌筑外墙，降低运输碳排放，与山地景观融合
  • 雨水收集系统：屋面坡度导流至地下储水槽，用于冲厕及场地灌溉
  • 紧凑平面布局：56㎡内整合男女厕、无障碍间及设备间，功能效率极高
启示：极小建筑同样可以成为生态技术的集成示范，在地材料 + 被动策略是低成本绿色建筑的可复制路径
