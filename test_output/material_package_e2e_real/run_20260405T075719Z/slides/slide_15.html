四至边界
  • 东侧：紧邻城市次干道，退界约3m，需控制噪声与视线遮挡
  • 西侧：现状居住区围墙，日照间距受限，立面开窗需审慎
  • 南侧：沿主干道展开，为主要人流界面与形象展示面
  • 北侧：绿化隔离带，可借景但不可占用
用地约束
  • 用地面积约300㎡，属高度紧凑型地块
  • 建筑密度上限60%，可建基底约180㎡
  • 限高24m，可利用垂直空间叠层布局
  • 消防通道需保证4m净宽，环绕可达
交通可达
  • 500m范围内覆盖地铁站点，公共交通便利
  • 南侧主干道设公交站，步行约2分钟可达
  • 机动车出入口建议设于东侧次干道，避开主界面
