Planning Analysis

上位规划解读

基于十堰市总体规划与张湾区控制性详细规划，梳理人民广场片区核心控制指标，明确规划刚性约束与弹性空间，为设计策略提供依据。

设计机遇

  • 低容积率为开放空间与景观渗透提供充裕余地
  • 文化功能定位契合城市公共文化服务体系升级需求
  • 高绿化率要求可转化为生态肌理设计特色
  • 建筑限高利于塑造舒展水平体量与城市天际线协调
控制指标十堰市总体规划张湾区控规
功能定位市级公共文化服务中心，承载文化展览、市民活动等复合功能片区文化核心节点，兼顾社区服务与城市形象展示
用地性质公共管理与公共服务用地（A类）文化设施用地（A2）
容积率片区整体 FAR 不超过 1.2地块 FAR 不超过 1.0（刚性上限）
建筑限高城市风貌协调区，建议 24m 以下控高 24m，局部标志性节点可适当突破
绿化率公共设施用地绿化率不低于 30%绿化率不低于 35%，鼓励屋顶绿化
建筑密度不超过 40%不超过 35%
退线要求主干道退线不少于 15m北侧人民路退线 15m，其余不少于 10m

数据来源：《十堰市城市总体规划（2016-2030）》及《张湾区控制性详细规划》公示文件
