Reference Case23

大唐不夜城「城东」公厕

Sangu Design / Xi'an, China
面积200 m²
年份2019
设计三谷设计
类型文化公建

项目位于西安大唐不夜城核心街区，以唐代建筑意象为设计母题，将公共卫生间转化为具有文化展示功能的城市微型空间，实现市政基础设施与历史文脉的深度融合。

设计策略分析

在地文化符号植入

提取唐代建筑的坡屋顶、斗拱、格栅等经典元素，以现代材料与简化手法重新演绎。外立面采用铝板格栅模拟传统木构肌理，既呼应不夜城唐风街区整体风貌，又避免了简单的形式复制。

空间品质化策略

内部引入自然采光天窗与庭院绿化，打破公厕封闭阴暗的刻板印象。动线设计融入无障碍通道与母婴室，服务标准对标商业综合体，提升公共设施的使用尊严感与城市形象。

场景融合与夜间表达

建筑夜间通过灯光系统呈现暖色调光影效果，与大唐不夜城的夜游经济场景无缝衔接，使功能性建筑成为街区景观节点，日均服务游客超过3000人次。

对本项目启示

  • 十堰在地文化（武当山道教建筑、汽车工业遗产）可转化为公共设施的视觉语言，增强场所认同
  • 公共空间品质化是提升城市文化形象的低成本高回报路径，功能建筑同样承载文化叙事
  • 夜间灯光与景观联动策略可借鉴，激活项目周边夜间活力与文旅价值
