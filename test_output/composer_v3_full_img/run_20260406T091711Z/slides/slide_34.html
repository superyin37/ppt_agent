CONCEPT 02 / AERIAL VIEW
方案二

隐山栖水

鸟瞰总览

覆土屋顶花园 — 种植本土耐旱植被，与广场绿地无缝衔接
下沉式入口庭院 — 引导人流自然过渡至地下展厅
雨水收集系统 — 屋面径流汇入景观水池循环利用
景观缓坡 — 建筑体量消隐于地形之中
方案二「隐山栖水」以覆土建筑策略将文化馆主体嵌入场地地形，屋顶花园与城市广场形成连续绿色界面，建筑轮廓消隐于景观之中。下沉入口通过缓坡与台阶引导访客进入半地下展览空间，营造由明入暗的仪式感。屋面雨水收集系统将径流导入下沉庭院景观水池，实现水资源的场地内循环，年节水量预计可达常规方案的 40%。整体设计呼应十堰山水地貌，实现建筑与自然的深度共生。
