Reference Case 01西安大唐不夜城公厕「城东」
城东 · 三谷设计2019200 m²
地点：西安大唐不夜城
建筑师：三谷设计
类型：文化公共设施
案例分析要点
01
唐风立面语言转译
以现代材料重构唐代建筑的坡屋顶与斗拱意象，将传统符号抽象为几何线条，避免直白复古，实现文化辨识度与当代审美的平衡。
02
文化场景与功能融合
200m²空间内整合卫生间、休息廊、文化展示墙等复合功能，将市政基础设施升级为街区文化体验节点，提升公共空间品质。
03
材料与光影策略
运用深色金属板与木纹格栅形成虚实对比，利用格栅过滤自然光线营造内部氛围，材料肌理呼应周边历史街区的质感基调。
04
城市界面处理
建筑体量压低、退让，以半开放灰空间过渡室内外，减少对步行街视觉的压迫，使公共设施自然融入大唐不夜城的游览动线。
▸ 对本项目的启示
十堰文化建筑可借鉴「城东」的文化符号抽象手法——将地域传统元素转化为现代建筑语言，而非直接复制。同时，复合功能植入与材料肌理叙事策略，可为本项目的公共空间文化表达提供有效路径。
