Reference Case 03

十堰草莓园活动馆

体块错落灰空间预制混凝土在地建造
建筑师小写建筑事务所
建筑面积620 m²
项目地点十堰市，中国
建成年份2019

设计特征

活动馆以多个方盒体块错落组合，形成丰富的灰空间与半室外活动场所。预制混凝土墙体搭配大面积玻璃幕墙，在保持结构质朴感的同时引入充足自然光线，实现室内外空间的柔性过渡。

对本项目的启发

  • 体块错落策略：通过不同高度与进退关系的体量组合，消解建筑体量感，形成尺度宜人的公共空间序列，可借鉴于本项目文化建筑的空间组织
  • 灰空间营造：檐下、连廊、架空层等半开放空间丰富了建筑与自然的交互界面，适合十堰湿润气候下的室外活动需求
  • 材质组合逻辑：预制混凝土的工业肌理与大面玻璃的通透感形成对比，可作为本项目立面材料策略的重要参照——混凝土体现文化沉稳，玻璃实现生态采光
  • 在地建造经验：同处十堰地区，其对当地施工条件、材料供应链的适应性策略具有直接参考价值
