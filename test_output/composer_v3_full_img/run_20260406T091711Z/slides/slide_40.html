项目名称：Material Package E2E
项目地点：湖北省十堰市
建筑类型：文化类建筑
编制日期：2024年度
SECTION 01
项目概况
  • 项目选址位于湖北省十堰市核心文化片区
  • 总建筑面积约 300 m²，地上一层为主
  • 用地容积率控制 ≤ 1.0，建筑密度 ≤ 40%
  • 建筑高度不超过 12m，与周边环境协调
SECTION 02
功能配置要求
  • 设置第三卫生间，满足特殊人群使用需求
  • 全流线无障碍设计，含坡道、扶手、盲道
  • 公共展示空间不少于总面积 40%
  • 配置管理用房、设备间及储藏空间
SECTION 03
技术指标要求
≥60%被动节能率
300m²建筑面积
1.0容积率
≤12m建筑限高
  • 采用被动式节能策略，节能率不低于 60%
  • 优先选用本地生态材料与可再生材料
SECTION 04
设计风格要求
  • 融合现代简约与传统文化元素
  • 体现工业肌理与自然生态的对话关系
  • 外立面注重材料质感，推荐清水混凝土、木材、石材
  • 室内空间强调留白与光影变化，营造沉静氛围
