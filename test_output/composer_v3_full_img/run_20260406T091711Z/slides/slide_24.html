Trado 生态公共厕所

MOL Arquitectura · Trado, Spain · 2018 · 56 m²
Reference Case 02

覆土被动节能策略

利用覆土屋面与自然材料实现极低能耗，为十堰文化建筑的生态设计提供可复用的技术路径。

56 m²
建筑面积
>60%
能耗降幅
100%
在地材料
  • 覆土屋面：300mm 种植土层覆盖混凝土壳体，夏季隔热冬季保温，全年空调负荷降低 60% 以上
  • 自然通风体系：利用地形高差形成贯穿式通风廊道，减少机械排风依赖
  • 在地石材与木材：主体采用当地花岗岩砌筑 + 栗木结构，运输碳排趋近于零
  • 雨水回收利用：覆土层自然滞蓄雨水，经过滤后用于冲洗与灌溉

对本项目启示：十堰地处秦巴山区，气候湿润、石材资源丰富，可借鉴覆土策略降低文化建筑运营能耗，同时以在地材料强化建筑与场地的共生关系。
