双核文化IP驱动：车城工业遗产与武当山水美学共塑建筑基因
IP 01
车城文化 — 东风汽车工业遗产
十堰因车建城，东风汽车50余年的工业积淀形成独特的城市肌理：厂房尺度、钢构桁架、机械韵律构成集体记忆。工业遗存中的铆钉阵列、冲压曲面、齿轮咬合等视觉符号具有极高的空间转译价值。
可转化设计元素：锈蚀耐候钢立面模块 / 桁架式遮阳构件 / 冲孔铝板幕墙肌理 / 工业轨道景观动线
IP 02
武当文化 — 道教山水美学
武当山道教建筑群以"因山就势、道法自然"闻名，紫霄宫的轴线层叠、南岩宫的悬崖飞檐展现极致的山地建筑智慧。太极阴阳、云水纹样、飞檐曲线等意象蕴含东方空间哲学——虚实相生、动静互补。
可转化设计元素：曲面屋顶呼应山势轮廓 / 太极动线组织内庭院 / 云纹穿孔板光影 / 层叠退台模拟山体意象
