CJJ 14-2016гҖҠеҹҺеёӮе…¬е…ұеҺ•жүҖи®ҫи®Ўж ҮеҮҶгҖӢвҖ” дәҢзұ»еҸҠд»ҘдёҠе…¬еҺ•
01

з¬¬дёүеҚ«з”ҹй—ҙ

дәҢзұ»еҸҠд»ҘдёҠе…¬еҺ•йЎ»и®ҫзҪ®зӢ¬з«Ӣз¬¬дёүеҚ«з”ҹй—ҙпјҢй…ҚеӨҮжҲҗдәәеҸҠе„ҝз«ҘеқҗдҫҝеҷЁгҖҒе®үе…ЁжҠ“жқҶгҖҒе©ҙе„ҝжҠӨзҗҶеҸ°дёҺе‘јеҸ«жҢүй’®пјҢйқўз§ҜдёҚе°ҸдәҺ 6.5гҺЎгҖӮ

CJJ 14-2016 В§5.3.1 / В§6.2.4
02

ж— йҡңзўҚи®ҫж–Ҫ

еҮәе…ҘеҸЈеқЎйҒ“еқЎеәҰвүӨ1:12пјҢйҖҡйҒ“еҮҖе®ҪвүҘ1.5mпјӣж— йҡңзўҚеҺ•дҪҚй—ЁеҮҖе®ҪвүҘ0.9mпјҢеҶ…йғЁеӣһиҪ¬зӣҙеҫ„вүҘ1.5mпјҢй…Қ L еһӢжү¶жүӢдёҺдҪҺдҪҚжҙ—жүӢзӣҶгҖӮ

CJJ 14-2016 В§5.4 / GB 50763
03

йҖҡйЈҺйҷӨиҮӯзі»з»ҹ

иҮӘз„¶йҖҡйЈҺжҚўж°”ж¬Ўж•°вүҘ5ж¬Ў/hпјҢдёҚж»Ўи¶іж—¶йЎ»и®ҫжңәжў°йҖҡйЈҺпјӣдәҢзұ»д»ҘдёҠе…¬еҺ•еә”й…ҚзҪ®зӢ¬з«ӢйҷӨиҮӯиЈ…зҪ®пјҢиҮӯж°”жө“еәҰиҫҫж ҮеҖјвүӨ20пјҲж— йҮҸзәІпјүгҖӮ

CJJ 14-2016 В§7.1.2 / В§7.3.1
04

иҠӮж°ҙдёҺз”ҹжҖҒиҰҒжұӮ

еӨ§дҫҝеҷЁеҶІж°ҙйҮҸвүӨ6L/ж¬ЎпјҢе°ҸдҫҝеҷЁвүӨ3L/ж¬Ўпјӣйј“еҠұйҮҮз”Ёдёӯж°ҙеӣһз”ЁгҖҒйӣЁж°ҙж”¶йӣҶеҸҠе…үдјҸдҫӣиғҪзӯүз»ҝиүІжҠҖжңҜпјҢеҘ‘еҗҲжө·з»өеҹҺеёӮе»әи®ҫеҜјеҗ‘гҖӮ

CJJ 14-2016 В§7.2 / GB 25502
05

еҺ•дҪҚй…ҚжҜ”ж ҮеҮҶ

з”·еҘіеҺ•дҪҚжҜ”дёҚдҪҺдәҺ 1:1.5пјҲе•Ҷдёҡж–ҮеҢ–еҢәе»әи®® 1:2пјүпјӣдәҢзұ»е…¬еҺ•з”·еҺ•дҪҚвүҘ4дёӘгҖҒеҘіеҺ•дҪҚвүҘ6дёӘпјҢй«ҳеі°ж—¶ж®өжҺ’йҳҹж—¶й•ҝвүӨ5minгҖӮ

CJJ 14-2016 В§4.2.3 / В§4.3
06

е»әзӯ‘дёҺзҺҜеўғиһҚеҗҲ

е…¬еҺ•еӨ–и§Ӯеә”дёҺе‘Ёиҫ№е»әзӯ‘йЈҺиІҢеҚҸи°ғпјӣж–ҮеҢ–зұ»йЎ№зӣ®йЎ»е…јйЎҫең°еҹҹж–Үи„үиЎЁиҫҫпјҢйј“еҠұйҮҮз”Ёең°ж–№жқҗж–ҷдёҺз”ҹжҖҒеҢ–еұӢйқўпјҢйҷҚдҪҺи§Ҷи§үзӘҒе…Җж„ҹгҖӮ

CJJ 14-2016 В§3.0.5 / ең°ж–№еҜјеҲҷ
