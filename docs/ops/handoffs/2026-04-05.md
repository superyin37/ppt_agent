## 6. 注意事项 / 陷阱

### 运行环境
- **`.env` 的 `DATABASE_URL` 是 docker 网络地址(`@db:5432`)**,host 运行脚本必须 override 到 `@localhost:5432`
- **Windows Celery** 必须 `--pool=solo`
- **先启动 db/redis**:`docker compose up db redis -d`

### 开发陷阱
- 不要再引用 `agent/graph.py`(已废弃,见 [../decisions/ADR-002](../decisions/ADR-002-celery-over-langgraph.md))
- Composer HTML 模式的 `spec_json` 结构与结构化模式完全不同,处理时必须检测 `html_mode` 字段
- Secret 不要写进文档或提交

### 产品边界
- 一期**只做 PDF**,不做 PPTX(`python-pptx` 已装但不实现)
- 上游"移动端素材采集"不在范围
- 目标是 `test_material/project1` 这种本地素材包,不是真实移动端产品
